##  Sample Interview Questions

**Q1:** What is ETL?
 Extract, Transform, Load process for data pipelines

**Q2:** Difference between INNER JOIN and LEFT JOIN?
 INNER JOIN returns matching rows, LEFT JOIN returns all left table rows

**Q3:** What is a data warehouse?
 System for storing and analyzing structured data

**Q4:** What is Apache Spark?
 Distributed data processing engine
